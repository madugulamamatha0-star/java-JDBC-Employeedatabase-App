# java-JDBC-Employeedatabase-App

    A console-based Java application that connects to a MySQL or PostgreSQL database using JDBC and performs full CRUD operations on an Employee table.

##Obective

->Connect a Java application to a relational database (MySQL/PostgreSQL).

->Perform Create, Read, Update, Delete (CRUD) operations.

Use:

1.Connection

2.PreparedStatement

3.ResultSet

##Features:
1. Seamless Database Connectivity
   
2.  Add Employees with Ease

3.  Instant Employee Lookup

4.  Smart Record Updates
